Application Layout

The main application layout is initialized on the start. By default all panels are defined, but only 3 are visible. You can show/hide additional panels at any time. The default layout is defined in /app/core/conf/config.js. The layout is created using w2layout from w2ui library. See w2ui.com for more information.
