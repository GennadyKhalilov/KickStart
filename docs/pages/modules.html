Modules

Loading the Module

The module should be loaded with app.load() method. You need to register the module before it is loaded in /app/core/conf/modules.js. Then, load it in the following way: If the module was not loaded before, it will be loaded. When the module is loaded the main module function is executed for the first time, triggering init() function. If module was already loaded, the render() function will be called.

You can pass additional parameters to the module in the following way: Which will come as obj.params or app.modName.params inside module function. Please note that params should be deleted in the render function, otherwise they will still be present when you call the module from another place of your code.
