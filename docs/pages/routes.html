Routes

As a developer you can define routes (#/module/params) in your application, so user can open a particular module with a direct link. There is an object app.routes that works with modules.
app.route.set (path)
Sets a route without triggering a call to this module. Use this function when you want to change URL without triggering action.
app.route.parse ()
Returns an array with elements of the route. This method makes sure that extra slashes are handled propely.
app.route.go (path)
Sets a route and triggers a call to that module. This method is called internally when user changes a route in the URL.
