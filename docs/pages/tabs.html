Application Tabs

The main application tabs are initialized on the start. The default set of tabs s defined in /app/core/conf/config.js. The tabs are not enabled by default, you need to open /app/core/conf/config.js and find show.tabs propety and set it to true. The tabs are created using w2tabs widget from w2ui library. See w2ui.com for more information.

All events from tabs will be sent to app.action() function defined in /app/core/conf/action.js, which you may extend with your application logic.
