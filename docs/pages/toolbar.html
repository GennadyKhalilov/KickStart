Application Toolbar

The main application toolbar is initialized on the start. You may change it to fit the needs of your application in /app/core/conf/config.js. The toolbar is created using w2toolbar widget from w2ui library. See w2ui.com for more information.

All button click events from toolbar will be sent to app.action() function defined in /app/core/conf/action.js, which you may extend with your own code.
